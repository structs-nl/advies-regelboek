Het biedt een analyse die in verschillende implementaties kan worden gebruikt, zonder een aanname te doen van een bepaalde technische implementatiekeus. Zowel visueel ontwerpers, solution architecten en programmeurs kunnen hiermee aan de slag en kunnen direct gebruik maken van de domeinspecifieke kennis die in het model is gevat. Een groot voordeel van deze benadering is dat deze domeinspecifieke kennis niet alleen in een technische implementatie terechtkomen. Deze benadering komt de toetsbaarheid, aanpasbaarheid en herbruikbaarheid ten goede.
